# Meetup Winner Picker

Sélectionne aléatoirement un gagnant parmi les participants inscrits à une page Meetup.

## Fonctionnalités attendues
- Extraction automatique de la liste des participants à un événement Meetup.
- Tirage aléatoire d’un ou plusieurs gagnants.
- Interface simple ou script CLI.
